Doodle Frogs

Personal Project

This Project is called "Doodle Frogs" and was meant to be published as a full NFT Collection but was halted due to market crashes. Each frog has a unique variation of colour, background, expression, clothing and accessories. The side goal for this project was to bring awareness for a non-profit organization that helps endangered frog species around the world.

  • Client: Jordan Kassar
  • Category: Illustration Design

Magazine Ad

College Program Project

The client is a shoe company called Strike that is looking to advertise their brand new shoe called the Strike TurboCharge 4. The client added that they had requirements that must be met for each ad and to include specific logos and website information. Our design is a eye catching magazine ad that uses contrast and hierarchy to grab readers attention. This project was completed using the Adobe InDesign Program.

  • Client: Mohawk College
  • Category: Layout Design

Typographic Imagery

College Program Project

The objective for this project was to choose an object that is to be created only using type. With the creativity in mind and the little baby yoda that was popping up all over social media, I decided to create a image of baby Grogu from Star Wars The Mandalorian. Using only type, I was able to successfully create shadows, wrinkles, colour transitions and other such details. The overall artwork has around 3,000 words placed together to form the image of the young alien staring deeply into the your eyes. This project was fun and challenging but well worth it, as it is one of my favorite creatives so far.

  • Client: Mohawk College
  • Category: Illustration Design

Menu

College Program Project

A new Mexican restaurant in Hamilton called "ROJO" need a new menu design to reflect their contemporary dining experience. Using modern and simple visuals and layout, we were able to design a multi-page menu that features all dishes, drinks and sides including attention for the Fajitas that chef wanted us to include. This project was done using the Adobe InDesign Program.

  • Client: Mohawk College
  • Category: Layout Design

Beer Case

College Program Project

The main focus of this entire project was to create an original concept for a Beer Brand (product). The goal was to complete a full artwork for a beer case that is suited with a theme assisted by a company logo and beer name. I was able to create an original idea using the company name "Outbound" and based it around nature. A brown bear represented the beer name "Big Bear" and from there the artwork I designed was fitted around nature and being outdoors. In the end, the final creative wrapped around the packaging, creating a seamless background that presented itself nicely.

  • Client: Mohawk College
  • Category: Packaging Design

Posters

College Program Project

Our objective with this project was to utilize our typography knowledge to create a logotype design for multiple posters. We were given the ability to create a logo for a fictional client, I ended up brainstorming and creating a fake band called "Tear of Fears." I came up with multiple concepts and found a design that I was able to utilize a theme out of for the posters. Using contrast and visuals, I was able to effectively pop the text off the page to bring to the viewers' attention. All information was presented well, and each poster had the bands vector details integrated into every design.

  • Client: Mohawk College
  • Category: Layout Design
